ინტერნეტიდან ჰოსტინგისთვის (Netlify Drop / tiiny.host) ატვირთე **მთელი საქაღალდე** (ყველა ფაილითა და ქვესაქაღალდით ერთად).

## 🎬 2. სად ჩავდო ვიდეოები

ვიდეოები ჩადე საქაღალდეში **`assets/videos/`** ზუსტად ამ სახელებით:

| ფაილი | რას შეესაბამება |
|---|---|
| `duda.mp4` | დუდას ვიდეოინტერვიუ |
| `lasha.mp4` | ლაშას ვიდეოინტერვიუ |
| `openair.mp4` | თბილისის Open Air-ის ვიდეო |

- ფორმატი: **MP4 (H.264)**.
- სახელი ზუსტად უნდა ემთხვეოდეს (ან შეცვალე გზა `data.js`-ში — იხ. ქვემოთ).
- ვიდეოებს უკვე აქვს `controls` და **ავტომატურად ხმაზე არ ირთვება**.

სხვა სახელი თუ გინდა, `data.js`-ში იპოვე `src` და შეცვალე, მაგ.:
```js
video: { src: "assets/videos/duda.mp4", ... }
```

## ✏️ 4. როგორ შევცვალო ავტორები, სათაური და ტექსტები

ყველაფერი **`js/data.js`**-შია. გახსენი და ჩაასწორე ბრჭყალებს შორის ტექსტი.

**სათაური და ავტორები** — ფაილის დასაწყისში, `meta`-ში:
```js
meta: {
  title: "ვინილის მხარე",                 // მთავარი სათაური
  titleMark: "B",                          // ოქროსფრად გამოკვეთილი ასო
  titleAccent: "ქართული მუსიკის რეალური ამბავი",
  subtitle: "...",                         // ქვესათაური
  authors: "ავტორები: გელა ზარქუა, ნუცა ნაგერვაძე",
}
```
ავტორების სახელი ასევე ავტომატურად ჩანს footer-ში (`footer.lines`).

**აბზაცების ტექსტი** — თითოეულ სექციას აქვს `paragraphs: [ "...", "..." ]`.
ცვლი პირდაპირ ბრჭყალებში. ახალი აბზაცის დასამატებლად ჩაამატე ახალი სტრიქონი მძიმით:
```js
paragraphs: [
  "პირველი აბზაცი.",
  "მეორე აბზაცი.",
  "ახალი აბზაცი აქ."
]
```

⚠️ **მნიშვნელოვანი:** ტექსტში ბრჭყალები დაწერე `„ "` (ქართული) ან `«»`.
ინგლისურ `" "`-ს ტექსტის შიგნით ერიდე — შეიძლება კოდი დაარღვიოს. სამაგიეროდ `„…“` უსაფრთხოა.

## 🎤 5. როგორ დავამატო ლევანის რეალური პასუხები

`data.js`-ში იპოვე `levani` → `qa`. თითოეულ კითხვას აქვს `q` (კითხვა) და `a` (პასუხი).
პასუხში ჩაანაცვლე `„[აქ ჩაიწერება ლევანის პასუხი]“` რეალური ტექსტით:

```js
qa: [
  {
    q: "როგორ ხედავთ ქართულ მუსიკალურ სცენას დღეს?",
    a: "„აქ ჩაწერე ლევანის ნამდვილი პასუხი.“"
  },
  ...
]
```

**ახალი კითხვის დამატება** — ჩაამატე ახალი ბლოკი მძიმის შემდეგ:
```js
{
  q: "შენი ახალი კითხვა?",
  a: "„ლევანის პასუხი.“"
}
```

## 💬 6. როგორ დავამატო ციტატები რესპონდენტებისგან

ციტატები რამდენიმე ადგილას ცხოვრობს `data.js`-ში:

- **დუდა / ლაშა** — თითო `quote:` ველი მათ ბარათში.
- **ლევანი** — `levani.quotes` (მასივი, quote card-ებად ჩანს):
  ```js
  quotes: [
    "„ლევანის პირველი ციტატა.“",
    "„ლევანის მეორე ციტატა.“"
  ]
  ```
- **Open Air** — `openair.quotes` (ფესტივალის მსმენელების ციტატები).
- **მსმენელები** — `listeners.cards`, თითო card-ს აქვს `text` (ციტატა) და `who` (ვინ თქვა):
  ```js
  cards: [
    { text: "„მსმენელის ციტატა.“", who: "სტუდენტი, 21 წლის" }
  ]
  ```

ახალი ციტატის დასამატებლად უბრალოდ ჩაამატე ახალი სტრიქონი/ბლოკი მასივში.

**გამოკითხვის სტატისტიკა** — `listeners.stats`-ში ჩაანაცვლე `—` რეალური რიცხვებით.
