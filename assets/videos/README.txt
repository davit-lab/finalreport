ვიდეოების ჩართვა — ინსტრუქცია
==============================

მთავარი გზა (რეკომენდებული): YouTube
------------------------------------
1. ატვირთე 3 ვიდეო YouTube-ზე — თითოეული "Unlisted" (ფარული) რეჟიმით.
   Unlisted ნიშნავს, რომ ვიდეო საჯარო ძიებაში არ ჩანს, მაგრამ ლინკით ხელმისაწვდომია.
2. თითო ვიდეოს ლინკი დააკოპირე (მაგ: https://youtu.be/AbCdEfGhIjK).
3. გახსენი ფაილი: js/data.js
4. იპოვე სამი ადგილი, სადაც წერია  youtube: ""  და ბრჭყალებში ჩასვი ლინკები:
      duda    →  youtube: "https://youtu.be/XXXXXXXXXXX"
      lasha   →  youtube: "https://youtu.be/XXXXXXXXXXX"
      openair →  youtube: "https://youtu.be/XXXXXXXXXXX"
5. შეინახე ფაილი და გახსენი index.html — ვიდეოები გამოჩნდება.

რატომ YouTube და არა პირდაპირ ფაილი?
------------------------------------
ნამდვილი ვიდეოები დიდია (208 MB + 124 MB + 516 MB ≈ 848 MB).
ასეთი მოცულობა სტატიკურ ჰოსტინგზე (GitHub Pages, tiiny.host, Netlify)
ან საერთოდ არ იტვირთება, ან ატვირთვისას წყდება. YouTube-ს ეს ლიმიტი არ აქვს,
საიტი კი პატარა რჩება და წამებში იტვირთება.
